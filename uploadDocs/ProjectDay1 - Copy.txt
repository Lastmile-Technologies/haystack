Project Day22


Για να ανεβάσετε σημεία μέσω excel, όπως βλέπετε το κουτί διαλόγου μπορείτε να πατήσετε στο κουμπί που αναγράφει “Ανέβασε” ή στο κουμπί που αναγράφει “Ανέβασε νέο σημείο αρχείο σημείων”. Αφού επιλέξετε ποιο excel θέλετε να ανεβάσετε, στον αντιστοιχητή που σας εμφανίζεται, αντιστοιχείτε τα πεδία με τις στήλες του excel που αντιστοιχούν. Για την ολοκλήρωση της διαδικασίας πατάτε στο κουμπί που αναγράφει “ΟΚ!” στο κάτω δεξιά τμήμα του αντιστοιχητή.

Εάν θέλετε να δημιουργήσετε καινούριο excel template ώστε να γίνεται σωστή και αυτόματη αντιστοίχιση με excel που περιέχει διαφορετικά πεδία, αφού έχετε πατήσει να ανεβάσετε excel, πατώντας στο πεδίο που βρίσκεται πάνω πάνω αριστερά στον αντιστοιχητή, σας δίνεται η δυνατότητα να φτιάξετε ένα καινούριο excel template.
