Dashboard Page1


Σε αυτή τη σελίδα υπάρχει η δυνατότητα παρακολούθησης στατιστικών σχετικά με την εταιρεία σας, η δημιουργία μιας καινούριας project day/ημέρα, και η εισαγωγή σε ένα project day που έχει την σημερινή ημερομηνία.

Για τη δημιουργία μιας καινούριας ημέρας/project day, μπορείτε να πατήσετε στο “+” που βρίσκεται ακριβώς κάτω από τη καρτέλα που αναγράφει ημέρα ημέρα στο πάνω δεξιά μέρος της σελίδας.

Για την εισαγωγή σε ένα project day που έχει την σημερινή ημερομηνία, πολύ απλά πατάτε πάνω στο project το οποίο θα φαίνεται στη ελαφρώς δεξιά πλευρά της σελίδας εκεί που αναγράφει “Live τώρα”.
