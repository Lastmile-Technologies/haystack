STOP POINT PAGE


Στο stop point page υπάρχει η δυνατότητα να δημιουργήσετε σημεία χειροκίνητα, να ανεβάσετε σημεία μέσω excel, να επεξεργαστείτε σημεία, να διαγράψετε σημεία, να διαγράψετε πολλά σημεία μαζί, να εκτυπώσετε voucher, και να κατεβάσετε αρχείο excel με όλα τα σημεία που βλέπετε στη λίστα.

Για να δημιουργήσετε σημείο χειροκίνητα, πηγαίνετε το ποντίκι σας στο κουμπί που αναγράφει “Νέες αποστολές” στη πάνω αριστερή γωνία του stop point page και πατήστε στο κουμπί που αναγράφει “Νέα αποστολή”. Αφού συμπληρώσετε τα στοιχεία σχετικά με το σημείο στο παράθυρο που σας βγήκε, πατάτε το “ΟΚ” για να ολοκληρώσετε τη διαδικασία.
