Project Day Overview Page2

Για να εισέλθετε σε ένα project day πολύ απλά κάνετε κλικ πάνω στο project day που επιθυμείτε.

Για να διαγράψετε ένα project day, πατάτε στις τρεις τελίτσες που βρίσκονται στο τέρμα δεξιά του project που θέλετε να διαγράψετε και πατάτε διαγραφή.

Για να διαγράψετε ένα project, πατάτε στις τελίτσες στη πάνω δεξιά γωνία της σελίδας που βρίσκονται ακριβώς κάτω από το εικονίδιο του χρήστη σας και πατάτε “Διαγραφή”.
