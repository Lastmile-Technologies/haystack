Project Page


Σε αυτή τη σελίδα υπάρχει η δυνατότητα δημιουργίας project και η είσοδος σε ένα ήδη δημιουργημένο project.

Για τη δημιουργία ενός project, πατάτε στο κουμπί που αναγράφει “Νέο” πάνω αριστερά στη σελίδα, ακριβώς κάτω από το λογότυπο της Lastmily. Στο παράθυρο που σας εμφανίζεται συμπληρώνετε τα στοιχεία που θέλετε να έχει το project, όπως όνομα, ποιο σταθμό/αποθήκη θέλετε να είναι συνδεδεμένο με το project, ποιοι ήδη δημιουργημένοι οδηγοί θέλετε να συμμετέχουν στο project και άλλες μικρορυθμίσεις. Για να ολοκληρώσετε τη διαδικασία της δημιουργίας, πατάτε το “ΟΚ!” που βρίσκεται κάτω κάτω.

Για την είσοδο σε ένα ήδη υπάρχων project, πολύ απλά κάνετε κλικ πάνω στο project στο οποίο θέλετε να εισέλθετε.
