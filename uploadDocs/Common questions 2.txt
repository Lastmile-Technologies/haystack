Συχνές ερωτήσεις 2:

Για διαγραφή σημείου όταν βρίσκεστε εδώ https://beta.lastmily.com/stops μπορείτε να πατήσετε στις τρεις τελίτσες που είναι στο τέρμα δεξιά ενός σημείου και να διαλέξετε διαγραφή. Εάν θέλετε να κάνετε μαζική διαγραφή, μπορείτε να τσεκάρετε τα κουτάκια των σημείων που θέλετε να διαγράψετε, να πατήσετε στις τρεις τελίτσες ενός σημείου που έχετε επιλέξει και να πατήσετε διαγραφή. Εάν βλέπετε τα σημεία σας στον χάρτη και θέλετε να τα διαγράψετε απο εκεί, απλά πατάτε πάνω στο σημείο που θέλετε να διαγράψετε και πατάτε τον μικρό κάδο που υπάρχει κάτω δεξιά στην οθόνη σας πάνω από το "ΟΚ!".

Για τη διαγραφή σημείων που δεν έχετε αιτηθεί μπορείτε να πάτε εδώ https://beta.lastmily.com/newShipmentsView, να πατήσετε τις τρεις τελείες στη δεξιά πλευρά του σημείου που θέλετε να διαγράψετε και να πατήσετε διαγραφή.

Οι οδηγοί αντιπροσωπεύονται με μικρές σαΐτες πάνω στον χάρτη. Αν πάτε το ποντίκι σας πάνω από κάθε σαΐτα θα δείτε τον οδηγό που αντιπροσωπεύει.

Για να κάνετε εκτύπωση των voucher, καθώς βρίσκεστε εδώ https://beta.lastmily.com/newShipmentsView , μπορείτε απλά να πατήσετε στο κουμπάκι που γράφει "Εκτύπωση".
